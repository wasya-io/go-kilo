# CHANGELOG

## [0.0.15] - 2024-02-17
### Changed
- アーキテクチャの大規模な改善
  - コードベースをクリーンアーキテクチャの原則に基づいて再構成
  - 単一責任の原則に基づいて各コンポーネントを分離
  - エディタの責務を明確に分割し、メンテナンス性と拡張性を向上

### Added
- テキストバッファ管理の分離と改善 (`buffer.go`)
  - Row構造体とその操作メソッドを専用ファイルに分離
  - 文字幅計算と位置変換ロジックの集約
  - バッファ操作の再利用性向上
